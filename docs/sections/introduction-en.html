Introduction back to ToC

The PROMESS ontology is being developed to organize and document the main entities involved in the study domain addressed by the project. The ontology focuses on the participant pathway through the study, including enrolment and grouping, follow-up visits, clinical and biological assessment, exposure information, imaging examinations, temporality, and clinical outcomes.

A documented ontology is useful for two complementary reasons. First, it provides a shared conceptual reference for researchers and collaborators who need a consistent understanding of the data and the terminology used in the project. Second, it provides the basis for a formal OWL implementation in which classes, properties, domains, ranges, cardinalities, and documentation metadata are made explicit and reusable.

The documentation follows the usual structure used for ontology publication on the Web. The intention is to provide a readable public page that presents the ontology metadata, the overview of its classes and properties, and the term-level descriptions generated from the ontology itself.
